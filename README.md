## Unno

* Unno 提供一种简单的层次数据的展示/存储/编辑方案。
* Unit 描述了数据的结构和约束（类似于 class），Node 描述了符合特定 Unit 的数据（类似于 class 的实例 instance）。
* 前端 html 页面内的 input 命名需要符合 Unit 结构。
